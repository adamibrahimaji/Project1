WEB BROWSER


Definisi Web Browser
Web Browser adalah suatu perangkat yang digunakan untuk menampilkan halaman-halaman web yang ada di internet


Adapun istilah-istilah yang sering muncul pada saat kita menggunakan web browser adalah sebagai berikut :
  1. web:halaman halaman web yang saling terhubung dalam suatu website
  2. Homepage : Halaman Awal ketika situs dimunculkan
  3. URL : Alamat unik pada halaman web
  4. WWW : Kumpulan dari beberapa dokomen elektronik
  5. Portal : Website yang menyediakan beberapa jenis layanan


Tiga jenis Web Browser yang sering dipakai
  • Internet Explorer

  • Netscape

  • Mozila
